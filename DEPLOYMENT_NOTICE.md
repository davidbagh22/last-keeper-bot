# Важное изменение развёртывания

Версия 4.0 использует постоянный диск Render:

- тариф сервиса: `starter`;
- диск: `last-keeper-data`;
- точка монтирования: `/var/data`;
- база: `/var/data/last_keeper.db`;
- автоматическая резервная копия создаётся при каждом запуске;
- владелец проекта может скачать копию командой `/backup`.

После синхронизации Blueprint Render может запросить подтверждение перехода с бесплатного тарифа на платный. Без этого постоянный диск не подключится, а файловая система останется временной.
