# HTML5-игра «Последний хранитель»

Файл `index.html` содержит самостоятельную мобильную игру для Telegram Web App.

## Состав

- 7 сюжетных решений;
- механика «эффекта бабочки»;
- 5 итоговых типов Хранителя;
- адаптивный интерфейс для телефона;
- работа без регистрации и базы данных.

## Публикация через GitHub Pages

Для размещения игры необходимо настроить GitHub Pages на папку `/webapp` либо перенести содержимое папки в ветку, используемую для публикации.

После публикации HTTPS-ссылку можно использовать в кнопке Telegram Web App или подключить через BotFather.
