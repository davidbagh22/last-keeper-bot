# Production v4

Версия 4.0 усиливает продукт по пяти направлениям:

1. **Надёжность** — база хранится на постоянном диске, при запуске создаются резервные копии, владелец может скачать базу через `/backup`.
2. **Понятность** — в меню появились «Состояние Архива» и «Моя коллекция».
3. **Драматургия** — бот показывает эхо последнего командного решения и связывает его с текущей главой.
4. **Ценность после игры** — участник собирает цифровую коллекцию культурных, научных и исторических фрагментов.
5. **Управление** — команда `/ops` показывает оперативную карту пяти команд по каждому дню программы.

Проверка `/health` теперь показывает путь к базе, наличие файла, размер и признак постоянного хранилища.
